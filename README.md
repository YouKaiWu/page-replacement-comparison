# Algorithm-comparison
- 比較四種 Page Replacement 演算法(FIFO, ARB, Optimal, Custom)的差異

---
## 環境

### OS
- Windows 11

### Java
- 版本：Java 24.0.2 (2025-07-15)

### Maven
- 版本：Apache Maven 3.9.11


---
### 執行方式
- note: 請使用 Bash 執行腳本、請確保 `run.sh` 有執行權限
- 在專案根目錄下執行以下指令：
`./run.sh` or `sh run.sh`
#### 執行輸出檔案
- log 檔存在 `run.log`
- 實驗結果存在 `./result` 底下的 excel 檔
